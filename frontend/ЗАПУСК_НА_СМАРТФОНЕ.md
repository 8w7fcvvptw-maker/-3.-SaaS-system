# Запуск проекта на смартфоне

## Быстрая инструкция

### 1. Убедитесь, что ПК и смартфон в одной Wi‑Fi сети

### 2. Узнайте IP‑адрес компьютера

**Windows (PowerShell):**
```powershell
ipconfig
```
Найдите «IPv4-адрес» (например, `192.168.1.100`).

**Mac/Linux:**
```bash
ifconfig | grep "inet "
```

### 3. Запустите проект

```bash
cd frontend
npm run dev
```

В консоли появится, например:
```
  ➜  Local:   http://localhost:5173/
  ➜  Network: http://192.168.1.100:5173/
```

### 4. Откройте на смартфоне

В браузере телефона введите:
```
http://ВАШ_IP:5173
```
Например: `http://192.168.1.100:5173`

---

## Возможные проблемы

| Проблема | Решение |
|----------|---------|
| Страница не открывается | Проверьте брандмауэр Windows — разрешите входящие на порт 5173 |
| Ошибка Supabase | Убедитесь, что в `.env` указаны правильные ключи Supabase |
| «Сайт недоступен» | ПК и телефон должны быть в одной Wi‑Fi сети |

## Мобильная адаптация

Проект адаптирован для мобильных экранов:
